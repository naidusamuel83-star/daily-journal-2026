# 🧠 Key Concepts Learned

## Arrays

* Collection of same type of elements
* Stored in continuous memory locations
* Fast access using indexing

---

## Time Complexity

### O(1) → Constant Time

* Direct element access

### O(n) → Linear Time

* Traversal operations

### O(n²) → Quadratic Time

* Nested loops

# 🔥 Important Patterns Learned

## 1. Traversal Pattern

```
for(int i = 0; i < arr.length; i++)
```

Used in:

* Sum
* Search
* Max
* Min

---

## 2. Two Pointer Pattern

```
int left = 0;
int right = arr.length - 1;
```

Used in:

* Reverse Array
* Palindrome Problems

---

## 3. Tracking Variable Pattern

```
int max = arr[0];
```

Used in:

* Largest Element
* Smallest Element
* Second Largest
* Second Smallest
